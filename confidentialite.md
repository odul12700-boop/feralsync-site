# Politique de confidentialité

*Dernière mise à jour : [DATE]*

## Ce que ce site collecte

Ce site est un site statique : il **ne collecte aucune donnée personnelle**,
n'utilise **pas de cookies de suivi** et n'intègre **aucun outil publicitaire**.

## Hébergement

Le site est hébergé par GitHub Pages. Comme tout hébergeur, GitHub peut
enregistrer des données techniques de connexion (adresse IP, type de
navigateur) dans ses journaux serveur. Ces données sont traitées par GitHub
conformément à sa propre politique de confidentialité.

## Liens externes et affiliation

Ce site contient des liens vers des sites tiers, notamment Amazon dans le cadre
du Programme Partenaires. Lorsque vous cliquez sur un lien d'affiliation, Amazon
peut déposer des cookies sur votre navigateur, régis par la politique de
confidentialité d'Amazon. Nous n'avons accès à aucune donnée personnelle vous
concernant via ce dispositif.

## L'application FeralSync

L'application mobile FeralSync fonctionne **entièrement hors-ligne** : les
données saisies (profil, horaires, préférences) sont stockées uniquement sur
votre appareil et ne sont transmises à aucun serveur.
